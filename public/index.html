Welcome to the Inventory Management API documentation.

endpoints

Products

  • GET /api/products - List all products (supports ?limit, ?offset, ?search, ?sort)
  • GET /api/products/low-stock - List products below reorder point
  • GET /api/products/:id - Get product details
  • POST /api/products - Create a new product
  • PUT /api/products/:id - Update a product (Full)
  • PATCH /api/products/:id - Update a product (Partial)
  • DELETE /api/products/:id - Delete a product

Stock Movements

  • GET /api/stock-movements - List stock movements
  • POST /api/stock-movements - Create a stock movement (Updates product quantity)

Suppliers

  • GET /api/suppliers - List suppliers
  • POST /api/suppliers - Create supplier
  • GET /api/suppliers/:id/products - View products by supplier
